Question 1: What are the different ways to select an element in the DOM?

Elements can be selected using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

We can change the HTML content (including tags) inside an element by using innerHTML and the visible text content inside an element, taking CSS styles into account by using innerText and all the text content inside an element, including hidden elements, without considering CSS styles by using textContent .

Question-3: What is event delegation in the DOM?

Event delegation is when you add one event listener to a parent element instead of each child element. This way, the parent handles events for all its children, even if they are added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event that happens on an element (like a button click) "bubbles up" to its parent elements. So, if a child element triggers an event, it will also trigger the same event on the parent, grandparent, and so on, until it reaches the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements with createElement(), add them using appendChild() or insertBefore(), and remove them with removeChild() or remove().
